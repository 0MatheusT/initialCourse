ESNext

Here will be presented some concepts about ESNext.

Spread/Rest Operator

          
            function sum(x, y, z) {
              return x + y + z;
            }
            
            const numbers = [1, 2, 3];
            
            console.log(sum(...numbers));
            // expected output: 6
            
            console.log(sum.apply(null, numbers));
            // expected output: 6
