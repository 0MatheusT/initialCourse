JavaScript: Control's Structures

Here will be presented some Control's Structures used in JS.

All structures was tested in console. To see the tests press f12.

IF/ELIF/ELSE Structure


            if (condition) {
              return
            } else if (condition) {
              return
            } else {
              return
            }

            switch (expression) {
              case value1:
                [break;]
              case value2:
                [break;]
              ...
              case valueN:

              default:

            }
